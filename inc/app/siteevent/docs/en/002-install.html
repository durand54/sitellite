SITEEVENT INSTALL INSTRUCTIONS

1. Install the necessary database tables:

$ mysql -p -u USER DBNAME < inc/app/siteevent/install/install-mysql.sql
(enter password when prompted)

2. Copy the event collection definition into the cms app:

$ cd /PATH/TO/SITELLITE
$ cp -R inc/app/siteevent/install/siteevent_event.php inc/app/cms/conf/collections/
