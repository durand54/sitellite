Boxes and forms

Here is the list of available boxes and their uses.

index

This box displays the list of used tags. The size of each tag is relative to its frequency of use. If tag parameter is given, it displays the list of items associated whit this tag.

Parameters

  • set: the name of the tag set we want to use.
  • tag: if specified, gives the list of items associated with this tag.

tags

This box displays the list of tags associated with the current page, and provides an add box to add new tags.

Parameters

  • set: the name of the tag set we want to use.
  • url: the url identifying the current page. It not specified, the page id will be used.
  • name: the title for the item. If not specified, use the title of the current page.
  • description: the description of the item. If not specified, use the description of the current page.

related

This box displays a list of related pages, based on tags shared between pages.

Parameters

  • set: the name of the tag set we want to use.
  • limit: the number of links to show.

edit form

This form provides an advanced view to edit item and associated tags. To be used, it needs set and url parameters.
