SITESTUDY INSTALL INSTRUCTIONS

1. Install the necessary database tables:

$ mysql -p -u USER DBNAME < inc/app/sitestudy/install/install-mysql.sql
(enter password when prompted)

2. Copy the case study collection definition into the cms app:

$ cd /PATH/TO/SITELLITE
$ cp -R inc/app/sitestudy/install/sitestudy_item.php inc/app/cms/conf/collections/
