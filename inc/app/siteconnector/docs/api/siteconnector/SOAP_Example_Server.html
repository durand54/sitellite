			$__dispatch_map
			 = array()			(line 45)
		
	


	

The dispactch map does not need to be used, but aids


the server class in knowing what parameters are used  with the functions.  This is the ONLY way to have  multiple OUT parameters.  If you use a dispatch map, you  MUST add ALL functions you wish to allow be called.  If  you do not use a dispatch map, then any public function  can be called from soap (in php4, we consider this to be  any function in the class unless it starts with underscore,  php5 support is not complete yet in this regard).  if you do not define in/out parameters, the function can be  called with parameters, but no validation on parameters will  occure.
