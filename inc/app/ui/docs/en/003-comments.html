Comments

The box is called ui/comments.

Here are the parameters you can pass to the box:

  • group (default: sitellite_page) : identify the kind of objects you want to comment.
  • item (default: current page id) : identify the item you want to comment.
  • approve (default: yes) : if yes, all comments are approved, if no, no comments are approved, if user, only comments from logged users are approved.
  • anon (default: false) : whether to allow getting comments from anonymous users.
  • readonly (default: no) : if yes, doesn't allow to add comments.

An item is identified from group + item.

You can style the comment box from /inc/app/ui/html/comments.css .
