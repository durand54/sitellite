Star rating system

The box is called ui/rating.

There is also a MailForm widget (ui.Widget.Rating) that acts like a select widget, but using stars. The widget is used in the above boxes. It is based on http://orkans-tmp.22web.net/star_rating.

Here are the parameters you can pass to the boxes:

  • group (default: sitellite_page) : identify the kind of objects you want to rate.
  • item (default: current page id) : identify the item you want to rate.
  • nstars (default: 5) : number of stars used to rate the item.
  • anon (default: false) : whether to allow getting rating from anonymous users.
  • readonly (default: no) : if yes, shows the average rating. If no, allows user to vote.
An item is identified from group + item. A vote is identified from group + item + username, where username is the current logged user username, of the user IP address if user not logged and anonymous votes are allowed.
