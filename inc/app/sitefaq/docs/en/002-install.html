Installation

SITEFAQ INSTALL INSTRUCTIONS

1. Install the necessary database tables:

$ mysql -p -u USER DBNAME < inc/app/sitefaq/install/install-mysql.sql
(enter password when prompted)

2. Copy the faq collection definition into the cms app:

$ cd /PATH/TO/SITELLITE
$ cp -R inc/app/sitefaq/install/sitefaq_glossary.php inc/app/cms/conf/collections/
$ cp -R inc/app/sitefaq/install/sitefaq_submission.php inc/app/cms/conf/collections/
